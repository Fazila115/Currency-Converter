# Currency Converter
A simple and responsive Currency Converter web application built using HTML, CSS, and JavaScript.
It allows users to convert currencies in real-time using an external exchange rate API.

# Features
- Convert between 100+ currencies
- Real-time exchange rates
- Dynamic country flags update
- Clean and modern UI
- Input validation (prevents empty or invalid values)
- Handles API errors gracefully

# Technologies Used
- HTML5 – Structure
- CSS3 – Styling & Layout
- JavaScript – Functionality

# Exchange Rate API –
https://latest.currency-api.pages.dev

# How It Works
- User enters an amount.
- Selects "From" and "To" currencies.
- Clicks Get Exchange Rate.
- App fetches real-time exchange data from the API.
- Converted amount is displayed instantly.

# API Used

Base URL:
https://latest.currency-api.pages.dev/v1/currencies

# How to Run
- Download or clone the repository
- Open index.html in your browser.
- No installation or backend required.
